# Implement-Various-Digital-Circuits-using-CMOS-Logic-in-Microwind


# CMOS VLSI Design Laboratory - 6th Semester
### Developed by: Ketan Shrirao

This repository contains a comprehensive collection of CMOS VLSI design experiments. 
The projects focus on the design, physical layout, and simulation of digital logic circuits using industry-standard EDA tools.

## 🛠️ Tools & Technologies
* **Software:** Microwind EDA Tool
* **Design Methodology:** λ-based CMOS Layout Design Rules
* **Key Focus:** Physical Layout Design, DRC (Design Rule Check), and Transient Analysis.

## 🧪 List of Experiments
The following circuits were implemented and verified through transient simulations:

### 1. Basic Logic Gates
* **CMOS Inverter:** Physical layout design to analyze propagation delay, rise/fall time, and voltage transfer characteristics.
* **2-Input NAND Gate:** Layout implementation using parallel PMOS and series NMOS networks.
* **2-Input NOR Gate:** Design using series PMOS and parallel NMOS configurations.
* **2-Input XOR Gate:** Realization of complex logic for arithmetic operations.

### 2. Arithmetic Circuits
* **Half Adder:** Physical layout of Sum (A ⊕ B) and Carry (A · B) outputs.
* **Full Adder:** Advanced combinational layout performing 3-bit addition.

### 3. Data Routing Circuits
* **2:1 Multiplexer:** Layout of a selection circuit based on a single select line.
* **4:1 Multiplexer:** Implementation of a four-input selection circuit using two select lines (S1, S0).

## 📂 Project Structure
Each experiment folder includes:
1.  **Layout File:** The final physical design (.MSK format for Microwind).
2.  **Simulation Results:** Waveforms verifying the timing and logic correctness.

## 🚀 Learning Outcomes
* Proficiency in **Microwind** for physical layout design and verification.
* Hands-on experience with **DRC (Design Rule Checking)** and **ERC (Electrical Rule Checking)**.
* Understanding of parasitic effects and power dissipation in sub-micron CMOS technology.
* Optimization of silicon area through efficient layout routing.
